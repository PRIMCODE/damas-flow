Raccourcis:
	`shift+clic`: selectionner un node ou un lien
	`l`: lier 2 nodes selectionnés (premier vers deuxième)
	`u`: supprimer de la database les liens selectionnés (unlink)
	`del`: effacer du viewport les nodes sélectionnés (la database est inchangée)
	`w`: editer les chemins vers les dossiers de travail
	`x`: exporter le graphe
	`d`: afficher/masquer ce texte (debug)
